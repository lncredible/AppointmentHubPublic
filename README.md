# AppointmentHub

The idea is to build a simple system for managing appointments, where you can create, update and keep track of bookings.

# Tech

- C#
- ASP.NET Core
- Entity Framework Core
- SQL Server

# Structure

The project is split into:
- Controllers
- Services
- Models
- Data layer

# Status

The project is not finished yet, but it's something I plan to continue working on and build out further over time.
